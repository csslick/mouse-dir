# mouse-dir
mouse direction background animation

## demo: https://csslick.github.io/mouse-dir/

## how to
```
  HTML
  <section id="title" class="mouse-dir" data-dir="">
    <h1>title</h1>
    <div class="bg">
    </div>
  </section>
```
```
  <script src="./mouse_dir.js"></script>
```

```
  var title = new Mouse_dir('#title');
  title.run();
```
